# World-Class Learning Loop — Brainstorm

## Problem

Qualified Delivery v2 can prove a strong floor, but it does not yet learn why a refined artifact
looks materially better. Without a controlled comparison and promotion rule, one attractive
reference can harden into a universal style superstition.

## User outcome

After repeated generation and art-direction loops, ease-design accumulates reusable evidence
about which decisions produce exceptional results, in which contexts, without weakening
correctness or accessibility.

## Product decision

Add a fourth `art-directed` variant and a new deterministic `learning-record` artifact. Keep floor
gates separate from six ceiling axes. Promote lessons only through repeated controlled wins or
explicit expert approval.

## Non-goals

- automated aesthetic truth;
- one universal world-class score;
- online model or network calls inside `ui`;
- silently rewriting the knowledge core from a single trial.

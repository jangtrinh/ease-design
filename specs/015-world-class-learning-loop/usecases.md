# World-Class Learning Loop — Use Cases

## UC-01 Controlled comparison

Given one benchmark and stable inputs, when a learning record is validated, then raw, enhanced,
qualified, and art-directed variants all exist and the evaluation is blinded.

## UC-02 Floor before ceiling

Given an art-directed artifact with a critical failure, when promotion is attempted, then the
lesson is rejected as premature regardless of its ceiling scores.

## UC-03 Evidence-backed promotion

Given three winning trials across two categories, when a lesson references those cases, then it
may be promoted. Given only one case, it remains a hypothesis or candidate.

## UC-04 Expert-owned hard rule

Given an explicit expert rule with a durable approval reference, when it is recorded, then it may
be promoted without fabricating repeated benchmark evidence.

## UC-05 Preserve context

Given a design choice that only works for one category, when recorded, then it is a contextual
recipe rather than a universal hard rule.

# Family: Retro Digital

Personas that revive **past digital aesthetics with modern craft** — Windows Vista
"Aero" glossiness and Y2K chrome/holographic finishes. Nostalgic optimism rendered
with present-day polish. Pick this family for playful, expressive landing pages,
portfolios, and ecommerce that lean into reflective, glossy, future-nostalgic looks.

**Personas in this family:** Neo-Frutiger Aero · Iridescent Chrome

See `persona-index.md` for the cross-family lookup table and auto-selection rules.

## Neo-Frutiger Aero

- **Slug:** `neo-frutiger-aero`
- **Family:** retro-digital
- **UI types:** landing, app
- **Density:** comfortable
- **Color mode:** light
- **Keywords:** aero, glossy, aqua, nostalgic, vista, retro, optimistic

**Philosophy** — Windows Vista-era optimism reborn: glossy aqua interfaces with
skeuomorphic joy.

### Aesthetic DNA

| Field | Direction |
|---|---|
| **Typography** | Rounded humanist sans (Fira Sans, Source Sans 3). Regular to medium weight. Friendly, approachable. Anti-aliased smoothness. Blue-tinted text. |
| **Color philosophy** | Aqua spectrum: sky blue (#87CEEB), ocean (#4682B4), white glass (#F0F8FF). Green (#32CD32) for positive actions. Glossy gradients: lighter top, darker bottom on all elements. |
| **Spacing** | Comfortable, familiar spacing. Standard padding (16–24px). Grouped components. Familiar OS-like proportions. |
| **Depth** | Glossy reflections (white gradient from top fading to 50%). Bevel effects on buttons. Drop shadows. Glass shelf reflections below cards. |
| **Borders** | Visible, friendly borders (1px #A0C4E8). Medium radius (8–12px), pill for buttons. "Jelly" border effects with inner highlight. |
| **Texture** | Glass and water-droplet effects. Lens flares. Sky/cloud background imagery. Glossy surface reflections on everything. |

**Interactions** — Glossy button depression on press (gradient inversion). Jelly
bounce. Glow effects. Familiar OS-like feedback.

**Layout** — Familiar OS/app patterns. Window-like containers with title bars.
Sidebar + content. Toolbar + workspace. Desktop metaphor.

**Anti-patterns** — NO flat design. NO dark themes. NO harsh/industrial aesthetics.
NO monochrome. Everything must feel OPTIMISTIC and GLOSSY.
**Avoid list:** flat design; dark themes; industrial aesthetics; monochrome palettes;
harsh contrasts.

## Iridescent Chrome

- **Slug:** `iridescent-chrome`
- **Family:** retro-digital
- **UI types:** landing, portfolio, ecommerce
- **Density:** comfortable
- **Color mode:** both
- **Keywords:** iridescent, holographic, chrome, metallic, reflective, y2k, glossy

**Philosophy** — Mirror-finish future nostalgia: iridescent, holographic, and chrome
surfaces evoking Y2K optimism with modern craft. Reflective material as identity.

### Aesthetic DNA

| Field | Direction |
|---|---|
| **Typography** | Geometric sans-serifs (Orbitron, Exo 2, Inter). Medium to bold weights. Metallic text effects via gradient text clipping. Uppercase for navigation and labels. |
| **Color philosophy** | Chrome spectrum: silver (#C0C0C0), platinum (#E5E4E2), mercury (#8B8B8B). Iridescent accents via multi-color gradients (pink→cyan→violet). Background can be white or dark — chrome adapts. |
| **Spacing** | Precise, measured. Moderate padding (20–32px). Symmetrical compositions. Everything aligned to an invisible matrix. |
| **Depth** | Mirror reflections (reflected gradient below elements). Glass-like depth with specular highlights. Metallic shadows with color tint. Chrome sheen gradients. |
| **Borders** | Thin reflective borders (1px with gradient or shimmer). Small-to-medium radius (6–12px). Some pill shapes for tags. Chrome edge effects. |
| **Texture** | Polished chrome gradients. Rainbow iridescent sheens. Holographic shimmer effects on surfaces. Clean, sterile backdrops with occasional foil accent. |

**Interactions** — Iridescent color-shift on hover (gradient angle rotation).
Reflective shimmer animations. Chrome glow effects. Liquid morphing transitions.

**Layout** — Symmetrical, balanced compositions. Floating chrome panels. Card grids
with reflective surfaces. Hero driven by reflective material rather than UI cards.

**Anti-patterns** — NO matte surfaces. NO warm earthy colors. NO serif fonts. NO
organic shapes. NO hand-drawn elements. NO vintage aesthetics that clash with chrome.
**Avoid list:** products needing very high corporate trust; highlight overload
creating visual noise; matte finishes; warm earthy tones.
